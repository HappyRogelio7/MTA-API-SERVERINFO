MTA API Documentation

Welcome to the documentation for the MTA API. This API provides access to server information and data.

Endpoints

GET

/api/:ip/:port

Retrieve server information based on IP and port.

Parameters

:ip - IP address of the server.

:port - Port of the server.

Response

The response is a JSON object containing server information.

{
    "name": "Server Name",
    "ip": "Server IP",
    "port": 12345,
    "password": 0,
    "players": 10,
    "maxplayers": 32,
    "keep": 0,
    "version": "1.0"
}
